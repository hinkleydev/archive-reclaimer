# Archive Reclaimer
A perl script to help store your data on CD media, designed for photos and old documents

In a world of an ever rising cost of living, cost of components, and very intrusive companies, storing data online seems less and less appealing. I want archive reclaimer to be a system which sorts data into folders based on criteria.

This is a personal challenge to myself. I have `Learning Perl, 3rd Edition` and that's all I intend to use for this development. Given the ever increasing amount of slop on the internet, I want to see how I fair with just a book, having no experience in Perl before.
